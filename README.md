# RS-Web_Development_Beginer

### Types of list
* unordered list
```
<ul>
    <li>like</li>
    <li>like</li>
    <li>like</li>
</ul>  
```
* Ordered list
```
<ol>
    <li>like</li>
    <li>like</li>
    <li>like</li>
</ol>
```
* Image tag 

Syntax:
```
<img src="image.address" width="size_X_px" height="size_y_px"/>
```

Height & width with percentage:
```
<img src="image.address" width="size_X%" height="size_y%"/>
```
- Alternative:
    ```alt="Text to show"```
* Anchor Tag:
```
<a href="https://youtube.com">Youtube</a>
```
